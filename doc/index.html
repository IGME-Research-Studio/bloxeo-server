    Storm Server

    Server for our Brainstorm App, a Sails application.

    Getting started

    Clone the repo.

        $ git clone git@github.com:IGME-Research-Studio/StormServer.git
    $ cd StormServer
    $ npm install
    $ npm install -g grunt-cli

    Commands

        # Lift the sails server
    $ npm start

    # Run the test suite
    $ npm test

    # Lift the sails server in debug mode
    $ npm run debug

    # Check the source code for stylistic correctness
    $ npm run lint
